# httpcat-java - Simple, stupid http.cat client made in Kotlin

Show HTTP status code as cute images of cats (or dogs!), with colors!

## Experience summary

Made this just after the Java version, a lot of things are similar (as it's still the JVM).

I used the same set of dependencies, and the syntax is nicer than Java. String interpolation is nice.

## Packages and versions

- Java 21
- Kotlin 1.9.22

- For CLI parsing: `info.picocli:picocli`
- To convert images to ASCII: `io.github.seujorgenochurras:image-to-ascii`

## How-tos

### Build

```sh
$ gradle build
```

### Usage

```sh
$ java -jar ./app/build/libs/app.jar 200
```
