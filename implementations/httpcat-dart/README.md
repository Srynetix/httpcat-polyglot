# httpcat-dart - Simple, stupid http.cat client made in Dart

Show HTTP status code as cute images of cats (or dogs!).

## Prerequisites

- nix
- nix-direnv
- devenv
- ...or just a recent version of Dart.

## Build

```sh
$ dart compile exe -o httpcat-dart ./bin/httpcat_dart.dart
```

## Usage

```sh
$ httpcat-dart 100
```
