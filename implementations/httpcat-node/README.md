# httpcat-node - Simple, stupid http.cat client made in Node

Show HTTP status code as cute images of cats (or dogs!).

## Prerequisites

- nix
- nix-direnv
- devenv
- ...or just a recent version of Node.

## Build

```sh
$ npm run build
```

## Usage

```sh
$ node ./dist/index.js 100
```
